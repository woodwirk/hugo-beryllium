Make a website with Hugo - Simple

2024-01-01

Elements

  • A computer with terminal/shell access
  • Web browser
  • Hugo executable
  • Hugo theme
  • Hugo quickstart

Instructions

  1. Install a version of Hugo on your computer
  2. Install git (or download a Hugo theme directly)
  3. Follow the Hugo quickstart guide
    • If you’re not using the git submodule approach, you will need to adapt the quickstart to your configuration.
  4. If you don’t want to use the hugo terminal commands, you can also create text file content directly.
