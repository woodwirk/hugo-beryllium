04 - Advanced - Make a website with Hugo

2025-01-01
hugo-features markdown content
Keys:
  • All of the features of this theme can be layered together -- front matter, taxonomy, shortcodes, etc.

Prepare

Hardware
  • Computer with shell/terminal
  • Internet access
Software
  • Web Browser: Any
  • Hugo Executable: v0.78 or later
  • Hugo Theme: hugo-beryllium
          (why not?)
Web
  • Quickstart: Hugo

Instructions

  1. Install a version of Hugo on your computer
  2. Install git (or download a Hugo theme directly)
  3. Follow the Hugo quickstart guide or the setup on the About page
    • If you’re not using the git submodule approach, you will need to adapt the quickstart to your configuration.
  4. If you don’t want to use the hugo terminal commands, you can also create text file content directly.
  5. Make this example a bundle and include images.
