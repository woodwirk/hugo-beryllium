This is an example website for rendering notes from plain text files using Hugo and the hugo-beryllium theme. It doubles as the example site some documentation.

The theme was forked from @jrutheiser/hugo-lithium-theme and @yihui/hugo-lithium. Additionally, it includes:

  • offline fuzzy searching from fuse.js (previously found in @woodwirk/hugo-lithium-search)
  • offline fonts (woff2)
  • offline photo gallery from PhotoSwipe
